🎓 Student Database Management System

A menu-driven Student Database Management System developed in C language that allows users to manage student records efficiently.
The system performs multiple operations such as inserting, updating, deleting, sorting, and reversing student records using structures and file handling for persistent storage.

🚀 Features

Add new student records
View all student records
Update existing student information
Delete student records
Sort student data
Reverse student records
Persistent data storage using file handling

🛠️ Technologies Used

C Programming Language
Data Structures
File Handling
Menu Driven Programming

📂 Project Structure
Student-Database-Management-System
│
├── main.c          # Main program file
├── database.txt    # File used for storing student records
└── README.md       # Project documentation
⚙️ How It Works

The program displays a menu-driven interface.

Users can select operations such as:
Insert student record
Display student records
Update student data
Delete student record
Sort records
Reverse records

All records are stored in a file, allowing the data to persist even after the program terminates.

▶️ How to Run the Program
Step 1: Compile the program
gcc main.c -o studentDB
Step 2: Run the program
./studentDB
